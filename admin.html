1. Scraping Google Maps

Pour collecter les médecins d'une spécialité et d'une ville spécifique en Algérie, procédez comme suit :

  1. Ouvrez Google Maps dans votre navigateur.
  2. Recherchez une requête précise (ex: pédiatre Batna).
  3. Ouvrez la console de développement (Appuyez sur F12).
  4. Copiez le script ci-dessous, collez-le dans la console et appuyez sur Entrée.
  5. Le script téléchargera un fichier JSON structuré directement sur votre machine.
